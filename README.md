* Data comes from official sources, more information [here](https://github.com/mayrop/datos-covid19in-mx)

## Installation
```R
if (!"remotes" %in% installed.packages()) {
  install.packages("remotes")
}
remotes::install_github("mayrop/datosmx")
```
